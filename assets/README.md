# Assets Directory

This folder is used to store static assets such as images, icons, and videos for the portfolio.

## Required Images
Please upload the following images to this folder for the Projects page to display logos correctly:

1. `uw.png` - Logo for University of Washington
2. `blue_origin.png` - Logo for Blue Origin
3. `umd.png` - Logo for University of Maryland

Once uploaded, the application will automatically link to `assets/filename.png`.
